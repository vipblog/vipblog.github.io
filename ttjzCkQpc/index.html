UVA12372 Packing for Holiday 题解

· 2020-06-06 · # 题解

description：

判断三个数是否都大于 202020，如果都大于则输出 bad，否则输出 good。

solution：

如题目描述。

code：

#include<cstdio>
using namespace std;
int main()
{
	int T;
	scanf("%d",&T);
	int cnt=0;
	while(T--)
	{
		cnt++;
		int a,b,c;
		scanf("%d%d%d",&a,&b,&c);
		if(a<=20&&b<=20&&c<=20)
		{
			printf("Case %d: good\n",cnt);
		}
		else
		{
			printf("Case %d: bad\n",cnt);
		}
	}
	return 0;
}
